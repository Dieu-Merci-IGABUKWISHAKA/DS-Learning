Bike Sales EDA Project
🔎 Overview

This project performs exploratory data analysis (EDA) on a bicycle product dataset to investigate pricing trends, brand and category distribution, and data integrity issues.

⚠️ Key Findings

52% of products have unmapped brands

100% of 2019 products lack brand mapping

Unmapped brands are associated with significantly higher prices

Brand data integrity issues materially affect pricing insights

📈 Analysis Includes

Data cleaning and validation

Brand and category pricing comparison

Model year trend analysis

Sensitivity analysis excluding 2019 and unmapped records

🧠 Skills Demonstrated

Data quality auditing

Foreign key validation

Bias detection in missing data

Sensitivity analysis

Business-oriented reporting
